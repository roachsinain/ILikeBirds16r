MCU型号对应的数字是十进制表示，遵守如下规则：

1.厂商和系列相同，则序号按顺序递增。

2.厂商相同，系列号不同，则系列号增1，序号从01开始。

3.mcu目录下新增代码后，要在b_config.h中同步更新配置。
